Context and Purpose

This is a small experimental web project based on react and bootstrap. This web page tries to mimic a basic shopping software: users can browse and buy the items they want by filtering and sorting them, and they can also remove items from their shopping cart.

Process

Hi-fi design

The functions

In this bakery store website, the user can choose to use the filter/sort at the left side.

  • Filter by bakery types
  • Filter by bakery dietary restriction
  • Sort by default setting (popularity)
  • Sory by price(low - high) or (high - low)

The users can also add/delete/reset the items in their cart.
